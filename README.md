# 💳 Add Chase Offers

Bookmarklet that walks the Chase Offers grid and adds every available offer to your card.

## 🚀 Install

**Drag:** open the [install page](https://julien-mo.github.io/add-chase-offers/) and drag the green button onto your bookmarks bar.

**Manual:** create a new browser bookmark named "Add Chase Offers" and paste the contents of [bookmarklet.txt](bookmarklet.txt) as its URL.

## 📖 Use

1. Log into Chase and open the [Offers page](https://secure.chase.com/web/auth/dashboard#/dashboard/merchantOffers/offerCategoriesPage?offerCategoryName=ALL) (the grid of tiles).
2. Click the bookmark. The page will bounce between each offer detail and the grid until every available offer has been added.
